# Stage 4 - Human Consolidation

Status: not yet executed.

This stage is not an autonomous LLM generation cycle. It records how researchers select, merge, reformulate, discard, or defer candidate directions produced in Stages 3a-3c.

It uses the previous LLM outputs as criticisable material, not as conclusions. If any LLM response presented its own "criteria", treat those criteria as model output, not as human criteria automatically adopted.

## Function

Turn exploratory material into a human-consolidated matrix of candidate directions.

The protocol itself should not become a candidate direction for the team. Keep only substantive directions connected to the team's trajectory.

## Consolidation criteria

- Connection to the team's documented trajectory.
- Explicit dependence on a theoretical anchor.
- Difference from mere description of previous work.
- Conceptual clarity.
- Relevance to the research domain.
- Potential contribution to quality, impact, or epistemological diversity.
- Feasibility with existing or realistically attainable competence.
- Risk of endogeneity, self-citation, ambiguity, fashion, or overclaiming.
- Need for external literature check or empirical evidence.

## Reporting rules

- In a report or article, present the consolidated matrix and the human reasons for keeping, merging, reformulating, discarding, or deferring directions.
- Reserve methodological discussion and appraisal of the process for the discussion section or equivalent reflective stage.
- Always distinguish LLM output, human interpretation, and what can be claimed with scientific prudence.
- Use verbatim LLM quotations only when analytically relevant; otherwise summarise.

## Notes on human responsibility

Record why each decision was made. The strongest output is not necessarily the longest list of directions. A smaller set of well-argued, critically bounded directions is preferable to a broad inventory of plausible-sounding topics.
