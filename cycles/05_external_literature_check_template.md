# Stage 5 - External Literature Check

Status: generic external-literature-check template.

This stage checks the human-consolidated directions against literature external to the team. It appraises plausibility and controls endogeneity. It does not replace a full literature review.

This is a human scholarly stage, not an LLM validation cycle. The LLM can help organise notes only when researchers verify the literature, scope, relevance, and final decision.

## Function

- Identify relevant external literature for each consolidated direction.
- Check whether the direction is substantive or merely terminological.
- Distinguish evidence from the team, evidence from external literature, and inference from the LLM.
- Decide whether the direction should be maintained, reformulated, discarded, or returned to an earlier stage.

## Evidence input

- Consolidation matrix from Stage 4.
- External literature search records.
- Inclusion and exclusion criteria for the literature consulted.
- Notes on recency, relevance, disciplinary scope, and methodological quality.

## External literature matrix

| Consolidated direction | External literature consulted | What supports plausibility | What problematises it | Decision |
|---|---|---|---|---|
|  |  |  |  | keep / reformulate / discard / return to Stage 3 |

## Decision labels

- keep;
- reformulate;
- discard;
- return to Stage 3a;
- return to Stage 3b;
- return to Stage 3c;
- require full literature review;
- require empirical evidence;
- require specialist consultation.

## Claims allowed after this stage

After this stage, the team may cautiously report:

- which candidate directions survived initial scrutiny;
- why they appear plausible;
- what remains uncertain;
- what external literature supports or problematises them;
- what further evidence is required.

The team should still avoid claiming that the LLM discovered, validated, or proved a scientific gap.

If this stage has not yet been completed, report the directions as human-consolidated candidates awaiting external literature appraisal, not as a finished agenda.
